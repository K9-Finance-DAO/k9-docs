# Claim

1. Click on "Claim" in the center menu\
   \
   ![](<../../.gitbook/assets/image (11).png>)
2. On this page you can claim your BONE tokens for previously executed applications. Please note that your requests are not immediately available for withdrawal, so withdrawals can only be made for “Ready” requests

<figure><img src="https://lh7-us.googleusercontent.com/docsz/AD_4nXchXl8ahX4WEdVeLHWdA2DuREC5y1DYUZIXSvkiHFpPHipPjIl1-BRTWmp_UJCpfiZ6MSmJTFvoWmq6fWq7SSgnh6tHMXuwtvFt1Yib8zbTPIRr420XJ7IJLrx03sM6H70ANVe3zmqMGpLCqP9bgcfeh2I?key=UHJy3JQkr0xZIlPVeyb1AA" alt=""><figcaption></figcaption></figure>

In the form on the left you will see:

1. **Reward** - The amount of all processed applications for which you can get BONE. This amount only includes “Ready” requests.
2. **Pending Amount** - The sum of all unprocessed applications. This amount only includes “Pending” requests.
3. **My Requests** - A list of your applications, which displays the amount and status of each application. In this list you can select those applications for which you want to receive BONE. You can select no more than 5 applications per request.

To claim your rewards you need to:

1. Go to the Claim tab.
2. In the "My Requests" field, select requests.
3. Click the “Claim” button in the “Reward” field and confirm the transaction in your wallet. Popup window displays the transaction progress:\
   ![](https://lh7-us.googleusercontent.com/docsz/AD_4nXfMNxbpf1RIFL0bTgHt-2-3rlQgHIpGh1RrQeB3jbZh7_B9eH8O2nXE-y0e9v5TdpMfYjbL_N3wECBS_hU_g6bwxP0-c5qV4YOc1VW6p_PVj3bwNtgsy_9GBj26rSJLqeO1cdGblgBMB5fRmbcnDzib0I4y?key=UHJy3JQkr0xZIlPVeyb1AA)&#x20;
4. Wait for the transaction to complete. BONE tokens will be credited to your wallet.\
