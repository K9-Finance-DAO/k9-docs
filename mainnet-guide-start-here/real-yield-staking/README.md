# Real Yield Staking

1. Click on "Real Yield Staking" in the left hand menu\
   ![](<../../.gitbook/assets/image (5).png>)
2.  This interface is for staking KNINE with optional locking to receive rewards. <br>

    <figure><img src="https://lh7-us.googleusercontent.com/docsz/AD_4nXcmn70ETv5lKzGdBQI7Sfivn1dtgSVwG5LWIeCNybEwTACNI-vz2clwe0itNO1krdP9HIHte02Nt7J-ZaPcFzzzDIt8iZZQwJszB2rIXnvKsNSNeZaVyi0tDV2Vj6JNFFswVgl8fNbF9LQrWzzt2xN0AOMB?key=UHJy3JQkr0xZIlPVeyb1AA" alt=""><figcaption></figcaption></figure>
3. Read the Real Yield Staking Rules/Preconditions:
   1. **Staking without Lockup:**&#x20;
      * Lockup period is not applied;
      * Base multiplier (x1) is applied for reward calculation;
      * Claiming rewards available anytime (when present);
      * Withdrawing available anytime.
   2. **Locked Staking:**&#x20;
      * Lockup period is applied - from 1 to 12 months:
        * min = 1 month;
        * max = 12 months;
      * Multiplier is applied for reward calculation: min > x1;
        * max = up to maxLockMultiplier;
      * Claiming rewards is available at anytime;
      * Withdrawing is available only after the lockup period ends.
   3. Preconditions
      1. Make sure you have KNINE tokens in your wallet;
      2. Make sure you have BONE tokens in your wallet to pay the transaction fee.
      3. Make sure your wallet is connected to Shibarium.
      4. Make sure your wallet is connected to the [K9 Finance Real Yield Staking dApp](https://app.k9finance.com/real-yield-staking/lock)&#x20;

Now you can choose from the following options (detailed guides available at links below)

* [Stake with Lock](stake-with-lock.md)
* [Stake without Lock](stake-without-lock.md)
* [Claim Rewards](claim-rewards.md)
* [Withdraw/Unstake](withdraw.md)
