# Claim All

When you want to only receive your accumulated rewards, you can use the “Claim All” button. By doing so you will receive all rewards from both “Stake” and “Zap” (if present):

1. Click the “Claim All” button
2. Confirm the transaction in your wallet. The popup will display the transaction status:\

3. Once the transaction is completed, the popup will display the transaction status:\
