# Farming

1. Click on "Farming" in the left hand menu\
   ![](<../../.gitbook/assets/image (12).png>)
2.  This page is for browsing farming pools. Here you can find pools to stake LP tokens to earn rewards.&#x20;

    <figure><img src="https://lh7-us.googleusercontent.com/docsz/AD_4nXeyQxP_2ne7zQo5WfQigurL7-4SonSr5MUmNwdcR88CXZsV6eRKLBB0Me1oTnbBFCEfTRpPm36BTU31RDGCcN1rVlJlzb-oR8PSctPIV9u5FcndgGFXUvHE9ioKBa6fsQcizZdQUkaom23owUxFlodQdKDC?key=UHJy3JQkr0xZIlPVeyb1AA" alt=""><figcaption></figcaption></figure>
3. Make sure to your wallet is connected and is on the Shibarium network
4.  Next, you will have access to the farming pools. Choose the one you prefer and click “Details” The farming pool page will open:

    <figure><img src="https://lh7-us.googleusercontent.com/docsz/AD_4nXeYMHs3b4EzOUlkLcLYUkXcaGbv1rf-ViBgSEdRwAIaRf8qJ71uiARXwVxbO4UcAhcixR2mDlmUcaDs3Oh-U_FiuPjRGceUUrx6Xxjk8jW9zbputwBc3U84cNrvMuPwNYTAeACjdeYva_pgHt21VXotQyY?key=UHJy3JQkr0xZIlPVeyb1AA" alt=""><figcaption></figcaption></figure>
5. Now you can choose from the following options (detailed guides available at links below)
   * [Stake/Unstake LP](stake-unstake-lp.md)
   * [Zap](zap.md)
   * [Claim All](claim-all.md)
   * [Bonus Rewards](bonus-rewards.md)
